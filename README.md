# Referral AI Platform

This MVP provides two dashboards:
- **Specialists** can update real-time waiting times for urgent, routine, and consultation visits, note whether they are accepting new patients, and add notes.
- **Family doctors** can search specialists by specialty, location, and sort by shortest wait time. Contact info and notes are visible.

All data is stored in the browser's `localStorage` for quick testing.

## Running locally
Open `index.html` in your browser. No build step is required.
